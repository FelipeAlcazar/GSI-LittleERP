# LittleERP

![LittleERP Logo](Assets/logo_horizontal.png)

## 📄 Descripción

LittleERP es una aplicación ligera de Planificación de Recursos Empresariales (ERP) diseñada para gestionar y optimizar los procesos empresariales. Incluye funcionalidades para la gestión de usuarios, gastos, ingresos y generación de informes en formato PDF.

## ✨ Características

- **👤 Gestión de Usuarios**: Registro, inicio de sesión y actualización de información de usuarios.
- **💸 Gestión de Gastos**: Agregar, eliminar y ver gastos.
- **💰 Gestión de Ingresos**: Agregar, eliminar y ver ingresos.
- **📄 Generación de Informes en PDF**: Generar informes en PDF para gastos e ingresos.
- **📊 Visualización de Datos**: Visualizar datos mediante gráficos.

## 🛠️ Tecnologías Utilizadas

- **C# y .NET**: Lenguaje de programación y framework principal.
- **UWP (Universal Windows Platform)**: Plataforma de la aplicación.
- **SQLite**: Gestión de bases de datos.
- **Syncfusion**: Generación de PDF y gráficos.
- **XAML**: Diseño de la interfaz de usuario.
- **Patrón MVVM**: Patrón arquitectónico para la separación de responsabilidades.


## 🚀 Comenzando

### 📋 Requisitos Previos

- Visual Studio 2019 o posterior
- .NET Core SDK
- Windows 10 SDK

### 🛠️ Instalación

1. Clona el repositorio:
    ```sh
    git clone https://github.com/yourusername/LittleERP.git
    ```
2. Abre el archivo de solución `LittleERP.sln` en Visual Studio.
3. Restaura los paquetes NuGet:
    ```sh
    nuget restore
    ```
4. Compila la solución:
    ```sh
    msbuild
    ```

### ▶️ Ejecutando la Aplicación

1. Establece `LittleERP` como el proyecto de inicio.
2. Presiona `F5` para ejecutar la aplicación.

## 📚 Uso

### 👤 Gestión de Usuarios

- **Registrar**: Crear una nueva cuenta de usuario.
- **Iniciar Sesión**: Autenticarse usando correo electrónico y contraseña.
- **Actualizar Perfil**: Modificar los detalles del usuario.

### 💸 Gestión de Gastos

- **Agregar Gasto**: Registrar un nuevo gasto.
- **Eliminar Gasto**: Eliminar un gasto existente.
- **Ver Gastos**: Mostrar una lista de todos los gastos.

### 💰 Gestión de Ingresos

- **Agregar Ingreso**: Registrar un nuevo ingreso.
- **Eliminar Ingreso**: Eliminar un ingreso existente.
- **Ver Ingresos**: Mostrar una lista de todos los ingresos.

### 📄 Generación de Informes en PDF

- **Generar Informe de Gastos**: Crear un informe en PDF de todos los gastos.
- **Generar Informe de Ingresos**: Crear un informe en PDF de todos los ingresos.
